# Blog App

A full-stack blog application built with Next.js, TypeScript, and PostgreSQL.
Users can create, edit, delete, and view blog posts with pagination and category filtering.

## Tech Stack

- Next.js
- TypeScript
- PostgreSQL
- Tailwind CSS
- Prisma
- Docker

## Features

- Create / Edit / Delete posts
- Pagination
- Category filter
- ISR (Incremental Static Regeneration)
- Authentication

## Installation

1. Clone the repository

git clone https://github.com/kritsada19/blog-app.git

2. Install dependencies

npm install

3. Create .env file

Add the following:

DATABASE_URL=your_database_url

4. Run development server

npm run dev
